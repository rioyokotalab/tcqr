# QR decomposition using TensorCore

## How to execute

1. Clone repository  
Clone with `--recursive`, because this project uses [cutf](https://gitlab.momo86.net/mutsuki/cutf) as submodule
```
$ git clone https://gitlab.momo86.net/mutsuki/tcqr --recursive
```

2. Build
```
$ make
```

3. Execute
```
$ ./tcqr.out
```
