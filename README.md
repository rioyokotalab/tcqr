# QR decomposition using TensorCore

## How to execute

1. Clone repository  
This project uses [cutf](https://gitlab.momo86.net/mutsuki/cutf), so clone with `--recursive`
```
$ git clone https://gitlab.momo86.net/mutsuki/tcqr --recursive
```

2. Build
```
$ make
```

3. Execute
```
$ ./tcqr.out
```
